How it works

  • Choose Government or Private. Gov mode injects gov domains & dorks.
  • Set scope: State / Central / All. State mode can be tuned per state (e.g. West Bengal).
  • Use Only PDF for official notifications; enable Deep / hidden to search notices buried in folders.
  • Toggle preset chips like site:gov.in, walk-in, contractual to refine.
  • Any change re-activates Build Query. Build, copy, then Search or Multi-Search to open multiple variant dorks.
